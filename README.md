

#  WEB-SCRAPPING ASSIGNMENT

In this Assignment I have used BeautifulSoup for Webscrapping.





## Features
This project focuses on extracting the useful content from the webpage such as : 
 - Product_Name
 - Brand_Name
  - Price (MRP)
 - Weight / Volume
  - Product_Image_URL
 - Product_URL
  - Reviews
 


## Requirements

 - BeautifulSoup
 - Requests Library
 - HTML.Parsar
 - Pandas
